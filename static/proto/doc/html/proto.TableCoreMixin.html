tableFromDictionary(dataDict, columnNames=None, sortable=True, tableId=None, expandable=False, visibleRows=6, presorted=None, **kwargs)¶

Render a table from data in dataDict. Each key in dataDict is a row title, each value is a list of values, each corresponding to the column given with columnNames.

If presorted is set to a number and tableId != None and sortable == True, that column will be presorted (using a hacky solution using jquery.
