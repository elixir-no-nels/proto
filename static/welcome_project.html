The unified web interface

The following tool have the unified interface of the methods available in the literature:

  • - genometriccorr described in paper [1]
  • - giggle described in paper [2]
  • - goshifter described in paper [3]
  • - IntervalStats described in paper [4]
  • - lola described in paper [5]
  • - stereogene described in paper [6]
The output of tools is a consolidated output of above methods.

Note:

This is a specialized version of The Genomic Hyperbrowser, focusing on new functionality for analyzing collections of genomic track. More information on the general HyperBrowser functionality can be found on the main project site.
